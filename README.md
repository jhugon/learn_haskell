# Learning Haskell

Requirements:

    sudo apt install libcairo2-dev

To run the programs run one of:

    cabal run helloworld
    cabal run makeCharts
    cabal run web
    cabal run hsplot

For `web`, then go to http://localhost:8081/isalive or
http://localhost:8081/chebyshev1st/5.234/10

## Tests

To run:

    cabal test
